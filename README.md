# Anthropic Emotion Research Explainer

An interactive, educational website explaining Anthropic's groundbreaking research on emotion representations in large language models (Claude). Built with modern web technologies and best practices for security, performance, and user experience.

## 🎯 Overview

This site provides an accessible, interactive explanation of the research paper:
> **"Emotion Concepts and their Function in a Large Language Model"** — Sofroniew, Kauvar, Saunders et al., Anthropic (2026)

## ✨ Features

### Core Content
- **8 Comprehensive Sections**: Why It Matters → Concepts → Representation → Experiments → Visual Lab → Implications → Limitations → Conclusion
- **Interactive Visualizations**: Emotion space maps, steering demos, danger meters, Elo comparisons
- **Live Emotion Activation**: Type messages and watch emotion vectors activate in real-time
- **Beautiful Typography**: Using Lora, Inter, and JetBrains Mono fonts

## 🚀 Deployment

### Deploy to Vercel (Recommended)

1. **Option 1: One-Click Deploy**
   - Push code to GitHub
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project" and select this repository
   - Vercel will auto-detect it's a static site
   - Deploy! 🎉

2. **Option 2: Vercel CLI**
   ```bash
   npm install -g vercel
   cd /path/to/project
   vercel
   ```

3. **Option 3: GitHub Integration**
   - Connect your GitHub account to Vercel
   - Select the repository
   - Auto-deploys on every push to `main`

### Local Development

No build step required! Just open `index.html` in your browser or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server

# Using Ruby
ruby -run -ehttpd . -p8000
```

Then visit: `http://localhost:8000`
